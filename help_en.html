GBG map editor for marking “support” provinces
for browser on-line game "FORGE OF EMPIRES"

Map editing consists of two stages: map setting and “support” provinces marking. Result map image can be copied to the clipboard, saved to a file, or published on the website imgbb.com.

Map setting.

First of all, it is necessary to setup guild headquarters (sign and select the color of the guild), and also check and set the number of spaces for buildings (for sieges) in the sectors.

To enter the editor, right-click on the desired sector. To set a province as a headquarters, enter the name of the guild, check the "headquarters" selection and choice the guild color. For other provinces, check the number of buildings for sieges (to select the number of places for buildings, click on the corresponding button). To save your changes, press SAVE (or ENTER on your keyboard). Cancel - CANCEL (or ESC). After completing the settings, the map can be saved to database of site jsonbin.io. You will get special ID of the map. This ID can be used by another player to continue editing. Otherway you can save map as file and sent to another player. Finally you can publish map as image on server imgbb.com to share it with your GBG partners.

Marking of “support” provinces.

To select Guild click on the headquarters. Then mark provinces that you plan to assign as supporters for the selected guild. Clicking again cancels the selection.

Map menu.

  • new - creating a new map (all provinces will have standard names)
  • save - write the map to jsonbin.io database (then you can send ID to another player to continue editing)
  • download - downloading a map from the jsonbin.io server (for editing)
  • clear - clear marking of all “support” provinces (headquarters and main settings won't be changed)

Image menu.

  • copy - copy the map image to the clipboard (you can immediately paste it into the messenger and forward it to your colleges for discussion)
  • save - write the map image to a jpg file on your computer
  • upload - upload the map image to the site imgbb.com (the resulting link to the map can be immediately inserted into a message in the “message center” right in the game)

File menu.

(extended)
  • save - write the map to a file
  • open - open map file to edit
