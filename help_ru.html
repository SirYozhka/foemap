Редактор карты секторов-опорников ПБГ
для браузерной онлайн-игры "FORGE OF EMPIRES"

Редактирование карты состоит из двух этапов: настройка карты и распределение опорников. Готовое изображение карты можно скопировать в буфер обмена, сохранить в файл или опубликовать на сайт imgbb.com.

Настройка карты.

Прежде всего необходимо распределить штабы гильдий по секторам (подписать и выбрать цвет штабов), а также проверить и установить в секторах количество мест под здания(осадки).

Для входа в редактор нажмите правую кнопку мыши на нужном секторе. Для установки сектора в качестве штаба введите название гильдии, отметьте выбор "штаб" и выберите цвет гильдии. Для остальных секторов проверьте количество зданий под осадки (чтобы выбрать количество мест под здания кликните на соответствующую кнопку). Для сохранения изменений нажмите SAVE (или ENTER на клавиатуре). Отмена - CANCEL (или ESC). После окончании настройки, карту можно сохранить в базу данных на сервер jsonbin.io. Вы получите специальный идентификатор (id) карты. Этот идентификатор можно переслать другому игроку для продолжения редактирования. Также карту можно сохранить в файл и переслать другому игроку.

Распределение опорников.

Кликните по штабу левой кнопкой мыши. Выбирайте сектора, которые планируете назначить опорниками для выбранной гильдии. Повторный клик отменяет выбор.

Меню "map".

  • new - создание новой карты (все сектора будут иметь стандартные названия)
  • save - запись карты на сервер jsonbin.io
  • load - открытие карты c сервера jsonbin.io (для редактирования)
  • clear - очистить выбор опорников (распределение штабов и осадок не изменится)

Меню "file".

(optional)
  • save - записать карту в файл (чтобы переслать другому игроку для продолжения редактирования)
  • open - открыть файл карты (для редактирования)

Меню "image".

  • copy - скопировать изображение карты в буфер обмена (можно сразу вставить в мессенджер и переслать согам для обсуждения)
  • save - записать изображение карты в файл формата jpg (тоже можно переслать, но позже)
  • upload - автозагрузка изображения карты на сайт imgbb.com (полученную ссылку на карту сразу можно вставить в сообщение в "центре сообщений" прямо в игре)
