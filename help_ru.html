для браузерной онлайн-игры "FORGE OF EMPIRES"

Редактирование карты состоит из двух этапов: настройка карты и распределение опорников. Готовое изображение карты можно скопировать в буфер обмена, сохранить в файл или сразу опубликовать на сайт imgbb.com.

Настройка карты.

Прежде всего необходимо распределить штабы гильдий по секторам (подписать и выбрать цвет штабов), а также проверить и установить в секторах количество мест под здания.

Для входа в редактор нажмите правую кнопку мыши на нужном секторе. Для установки штаба введите название гильдии, отметьте выбор "штаб" и выберите цвет гильдии. Чтобы выбрать количество мест под здания - просто кликните на соответствующую кнопку. Для сохранения изменений нажмите ENTER. Отмена - ESC. Просто переход к редактированию другого сектора не изменяет текущий. После окончании настройки, карту можно сохранить в файл и переслать другому согу для продоложения работы.

Распределение опорников.

Кликните по штабу левой кнопкой мыши. Выбирайте сектора, которые планируете назначить опорниками для выбранной гильдии. Повторный клик отменяет выбор.

Меню "file".

  • new - создание новой карты поля (все сектора будут иметь стандартные названия)
  • save - запись карты в файл, который можно переслать другому игроку для дальнейшего редактирования
  • open - открыть файл карты для редактирования
  • clear - очистить выбор опорников (распределение штабов и осадок не изменится)

Меню "image".

  • copy - скопировать изображение карты в буфер обмена (можно сразу вставить в мессенджер и переслать согам для обсуждения)
  • save - записать изображение карты в файл формата jpg (тоже можно переслать, но позже)
  • upload - автозагрузка карты на сайт imgbb.com (полученную ссылку на карту сразу можно вставить в сообщение в "центре сообщений" прямо в игре)
